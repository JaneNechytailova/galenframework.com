Page Layout Testing

Galen is a tool for layout testing of web applications. It uses Selenium to obtain objects on page and then verifies if they are aligned appropriately. Idea is simple: you tell Galen how a page should look like and Galen will check that

Responsive Design Testing

Galen is designed with responsivness in mind. It is easy to set up a test for different browser sizes. Galen just opens a browser, resizes it to a defined size and then tests the page according to specifications

Powerfull syntax

Galen is not just a tool. It is a special language with powerfull syntax which gives you the ability to express even the most complex layouts. And even more than that. Galen has a special syntax for parameterizing your tests in the way you want
@ mobile, tablet ----------------------------- comments width: 300px inside: screen 10 to 30px top right near: article-content > 10px right @ desktop ----------------------------- comments width: ~ 100% of screen/width below: article-content > 20px

Runs in Selenium Grid

As Galen uses Selenium you also get the complete advantage of Selenium Grid. You can perform cross-browser testing and always make sure that the small changes in your code do not break layout in all other browsers
