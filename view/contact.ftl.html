Contact

If you would like to ask question about Galen Framework you can do it on Galen Framework Google Group.

In case you have questions specific to project development or just would like to discuss its future improvements or any other technical details you can join the Galen Framework chat on Gitter

If you find a bug you can report it on GitHub issues. But first please see if somebody already posted similar issue. Please also provide all the details if possible:

  • Url to website which you tested with Galen
  • Error messages with complete stacktrace
  • Spec and test files to reproduce the issue
  • Command with which you ran your tests
